## 4a.11 函数调用语法

### 基本调用

```
函数名(参数1, 参数2, ...)
函数名(参数1; 参数2; ...)
```

- 参数用 `,`（逗号）或 `;`（分号）分隔
- 调用函数需要括号（0 参数函数除外）
- 空括号 `()` = 0（不是函数调用）
- 支持嵌套：`sin(cos(0))`

### 0 参数函数

```
now         → 当前日期时间
today       → 今天的日期
```

### 可变参数

```
gcd(a, b)           → 恰好 2 个参数
gcd(a, b, c, ...)   → 2+ 个参数（可变）
log(x) 或 log(x, base) → 1 或 2 个参数
```
