## 8.6 数据集

### 化学元素

数据集名称: `elements` [来源: pyqalculate_data/elements.json]

包含 118 种化学元素，属性包括：
- `symbol` — 元素符号
- `name` — 元素名称
- `number` — 原子序数
- `mass` — 原子质量
- `group` — 族
- `period` — 周期

**使用示例**:
```
element("Fe", "mass")  # 铁的原子质量
```

### 行星

数据集名称: `planet` [来源: pyqalculate_data/planets.json]

包含 10 个太阳系天体，属性包括：
- `name` — 名称
- `mass` — 质量
- `radius` — 半径
- `distance` — 距太阳距离

**使用示例**:
```
planet("earth", "mass")  # 地球质量
```
